FLEXBOX

Es un modelo de diseño del CSS3 que fue diseñado para ser mas adecuando ala creacion de sitios responsivos, lo cual permite usar menos codigo y tambien evitarse el uso del float. La idea proncipal del flexbox es basado al especio que se tiene alterar los anchos y altos y tambien posicionar mejor los elementos Se puede decir tambien que es modulo muy completo, que esta relacionado directamente con el contenedor del sitio

Presenta la siguiente Estructura:

Su composición es en contenedores llamados Padre e Hijo, en donde el Padre es El contenedor Flex y el Hijo los ELementos Flex

Podemos encontrar las siguientes Propiedades:

Nivel del Contenedor Flex: (Padre)

  1. Display
  2. Flex-direction
  3. flex-wrap
  5. flex-flow
  6. justified-content
  7. align-items
  8. align-content

Elementos Flex: /Hijo

  1. order
  2. flex-grouw
  3. flex-shrink
  4. flex-basis
  5. flex
  6. align-self

CSS-GRID

Es una especificación de W3C diseñada que proporciona un método de forma bi-dimensional en la creación de maquetación CSS, y ofrece la posibilidad de crear elementos de la página con el uso de lineas e columnas. También puede ser usado para obtener muchos estilos de maquetación. Una caracteristica de este método es la creación de layouts mediante la división de grandes regiones y de definir su relacion en medidas, posición de capas entre los diferentes componentes del diseño HTML.
